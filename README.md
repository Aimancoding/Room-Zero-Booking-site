# Room Zero - Escape Room Booking Website

## Project Description
Room Zero is a cyberpunk-themed escape room booking website. It features immersive storylines, multiple themed rooms, and an interactive booking system. The website provides information about various escape rooms with different difficulty levels and capacities.

## Features
- Interactive escape room showcase with 6 different themed rooms
- Dedicated About Us page with team information and statistics
- Customer reviews section with submission form
- Booking modal for room reservations
- Responsive design with cyberpunk aesthetic
- Glowing effects and modern UI

## Escape Rooms Available
1. **Victorian Detective Mystery** - Easy (2 Players)
2. **Haunted Mansion Escape** - Medium (4 Players)
3. **Spaceship Crash Landing** - Hard (6 Players)
4. **Time Traveler's Lab** - Extreme (8 Players)
5. **Floor Zero** - Customizable
6. **The Forgotten Facility** - Almost Impossible (8-12 Players)

## How to Run
1. Clone this repository or download the ZIP file
2. Extract all files to a folder
3. Open `index.html` in your web browser
4. Navigate through the website using the navigation menu
